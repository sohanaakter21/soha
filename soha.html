Course Title: Internet Concepts and Web Technology


Here is the course curriculum of subject INC.Please find this form the attachment & set your mind to gain success after completing this course.

  • Course Objective:
      1. Understand how web technologies interact, from the browser to the backend servers. 2. Gain familiarity with numerous languages. 3. Learn database design basics and SQL access. 4. Implement dynamic content web pages using two- and three-tier architectures. 5. Gain practical experience in a selected set of Web technologies. 6. Implement a web application with database content and a dynamic client.
  • Course Content:
      1. Basic Concepts of the Internet: Introduction to Internet and www, Architecture of www, Kinds of website, difference between website and webpage, Concept of URL, Website design Concept, Tools and analysis. 2. Web Designing Language: HTML, CSS, Bootstrap 3. Web Development: PHP, Javascript 4. MySQL Database: Introduction to database and SQL, retrieving data from MYSQL using PHP, Using PHP to manipulate data in MYSQL.
  • Learning Outcome:
      1. Basic concept of front-end development technologies including HTML, CSS, and JavaScript 2. Practical knowledge for design methodologies for PHP and responsive web design 3. User Experience of the web server works and provide knowledge for servicing client requests

  • Text Book:
      Beginning PHP5 By: Wankyu Choi, Dan squir, Dave W. Mercer, Allan Kent.
